Bar Graph

  • When is_trend_analysis == False, plots the total number of documents in the period of analysis for the most frequent items in a field using a horizontal bar plot.

  • When is_trend_analysis == True, plots the total number of documents in the period of analysis for the items with the highest average growth rate in a field using a horizontal bar plot.
